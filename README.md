# Projeto de Engenharia de Dados

## 📊 Objetivo
Realizar um processo de ETL (Extract, Transform, Load) e criar um dashboard no Power BI.

## ⚙️ Tecnologias
- Python (Pandas)
- Power BI
- GitHub

## 🔄 ETL
- Extração de dados CSV e JSON
- Transformação (limpeza, cálculo de métricas)
- Integração dos dados
- Geração de base consolidada

## 📈 Dashboard
- Receita por região
- Top produtos mais vendidos
- Evolução mensal das vendas

## 📁 Estrutura
- dados/
- scripts/
- output/
